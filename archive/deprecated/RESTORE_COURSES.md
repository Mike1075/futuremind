# 课程页面恢复计划

## 三个课程的共同设计模式

### 1. 倾听课程 (listening)
- **路径**: `/admin/courses/listening/page.tsx`
- **左侧**: 14天索引列表
- **右侧**: 文本内容编辑 (标题、原文、解读、冥想、练习) + 音频资料管理
- **保护**: 前14天不可删除，新增天数可删除
- **来源**: commit `af3df01`

### 2. 地球课程 (earth)
- **路径**: `/admin/courses/earth/page.tsx`
- **左侧**: 6个阶段索引列表
- **右侧**: 阶段内容编辑 (标题、纪录片、引导、知识点、提问、反思) + 视频资料管理
- **保护**: 前6个阶段不可删除，新增阶段可删除
- **来源**: commit `87ff28f`

### 3. PBL课程 (icarus/pbl)
- **路径**: `/admin/courses/pbl/page.tsx`
- **布局**: 3×4矩阵视图（模块×难度）
- **右侧**: 项目详情编辑（标题、描述、周计划、前置条件）
- **保护**: 前12个项目（3个默认模块）不可删除，新增模块可删除
- **来源**: commit `892885d`

## 恢复步骤

1. 从git历史恢复三个页面文件
2. 保持这三个独立页面的设计不变
3. 移除通用课程详情页中的"课程内容"标签（因为应该在各自的专属页面查看）
4. 构建并测试
5. 提交并推送
