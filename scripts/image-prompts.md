# 伊卡洛斯计划 - PBL项目封面图片生成Prompts

生成日期: 2025/11/5 19:41:26

## 使用说明

1. 复制每个项目的prompt
2. 粘贴到AI图片生成工具（Midjourney、DALL-E、Stable Diffusion等）
3. 生成图片并下载
4. 重命名为对应的项目ID.png
5. 上传到 public/images/project-covers/
6. 运行 `npx tsx scripts/update-project-covers-manual.ts` 更新数据库
