### 1.2 光粒子效果实现方式

**实现技术：纯几何体（圆形粒子）+ Canvas合成模式**

**关键技术点**：
1. **粒子绘制**：使用`ctx.arc()`绘制圆形粒子
2. **发光效果**：通过`ctx.globalCompositeOperation = 'lighter'`实现光粒子叠加发光
3. **颜色系统**：HSLA色彩空间，红色系（H=0-8°）

**核心代码**（`ConsciousnessTreeCanvas.tsx:62-76`）：
```typescript
// 设置发光效果
ctx.globalCompositeOperation = 'lighter'

// 绘制粒子
particles.forEach((p) => {
  ctx.beginPath()
  ctx.arc(p.x, p.y, p.size, 0, Math.PI * 2)
  ctx.fillStyle = p.color
  ctx.fill()

  // 增强发光效果（针对高亮粒子）
  if (p.color.includes('70%') || p.color.includes('72%')) {
    ctx.shadowBlur = 15
    ctx.shadowColor = p.color
    ctx.fill()
    ctx.shadowBlur = 0
  }
})
```

### 2.3 树干粗细（Thickness）控制机制

**位置**：`consciousnessTreeGenerator.ts:160-190`

**核心算法**：
1. **初始宽度**：由`trunkWidth`参数设定（默认12px）
2. **层级衰减**：每层分支宽度按0.7的指数衰减
3. **粒子层数**：根据宽度计算需要多少层并列粒子

**关键代码片段**：
```typescript
// 计算当前分支宽度（层级衰减）
const currentWidth = Math.max(
  trunkWidth * Math.pow(0.7, currentDepth - 1),
  0.5
)

// 绘制线条函数
const drawLine = (x1, y1, x2, y2, width, type) => {
  // 计算需要多少层粒子并列排布（根据宽度）
  const layers = Math.max(1, Math.floor(width / particleSize))

  // 在width范围内均匀分布layers层粒子（并列不重叠）
  for (let layer = 0; layer < layers; layer++) {
    const offset = (layer / Math.max(layers - 1, 1) - 0.5) * width
    const py = baseY + offset
    addParticle(px, py, type, 1)
  }
}
```

**粗细计算公式**：
```
当前宽度 = trunkWidth × 0.7^(层级-1)
粒子层数 = floor(当前宽度 / particleSize)
```

## 3. 视觉效果实现逻辑

### 3.1 生成方式：程序化生成（Procedural Generation）

树是**动态程序化生成**的，每次渲染都实时计算。不是预加载的静态模型。

**生成流程**：
1. 确定画布中心点作为树干起点
2. 递归生成树干和树冠（向上生长）
3. 递归生成根系（向下生长）
4. 根据层级和概率在末端添加叶子和果实

### 3.2 递归分支算法

**位置**：`consciousnessTreeGenerator.ts:192-234`

```typescript
const branch = (x, y, len, angle, currentDepth, maxDepth, isRoot = false) => {
  if (currentDepth > maxDepth) return

  // 计算终点坐标
  const endX = x + len * Math.sin((angle * Math.PI) / 180)
  const endY = y - len * Math.cos((angle * Math.PI) / 180)

  // 绘制当前分支
  drawLine(x, y, endX, endY, currentWidth, type)

  // 递归生成子分支（二叉树结构）
  const newLen = len * lengthDecay

  if (isRoot) {
    // 根系：使用rootSpread角度
    branch(endX, endY, newLen, angle + rootSpread, currentDepth + 1, maxDepth, true)
    branch(endX, endY, newLen, angle - rootSpread, currentDepth + 1, maxDepth, true)
  } else {
    // 树冠：使用branchAngle角度
    branch(endX, endY, newLen, angle + branchAngle, currentDepth + 1, maxDepth, false)
    branch(endX, endY, newLen, angle - branchAngle, currentDepth + 1, maxDepth, false)
  }
}
```

### 3.3 为什么根系只能分两叉？

**技术原因**：算法采用**严格的二叉递归**结构（Binary Tree）

**代码证据**（`consciousnessTreeGenerator.ts:225-233`）：
```typescript
if (isRoot) {
  const spread = rootSpread * (1 + Math.random() * 0.5)
  branch(endX, endY, newLen, angle + spread, currentDepth + 1, maxDepth, true)  // 左根
  branch(endX, endY, newLen, angle - spread, currentDepth + 1, maxDepth, true)  // 右根
} else {
  const spread = branchAngle
  branch(endX, endY, newLen, angle + spread, currentDepth + 1, maxDepth, false)  // 左枝
  branch(endX, endY, newLen, angle - spread, currentDepth + 1, maxDepth, false)  // 右枝
}
```

**设计说明**：每个节点固定产生2个子节点（+spread和-spread），无法产生3叉或更多分支。

## 5. 关键代码片段分析

### 5.1 递归深度控制树的密集度

**代码位置**：`consciousnessTreeGenerator.ts:236-240`

```typescript
// 生成树干（向上）
branch(startX, startY, trunkLength, 0, 1, depth, false)

// 生成根系（向下）
if (rootDepth > 0) {
  branch(startX, startY, trunkLength * 0.6, 180, 1, rootDepth, true)
}
```

**公式**：
```
总分支数 ≈ 2^depth （二叉树）
depth=10 → 约1024个分支
depth=14 → 约16384个分支（超密集）
```

### 5.2 粒子大小不随宽度变化（关键设计）

**代码位置**：`consciousnessTreeGenerator.ts:187`

```typescript
// 粒子大小固定为particleSize（不再乘以倍数）
addParticle(px, py, type, 1)  // ← sizeMultiplier固定为1
```

**设计理念**：
- **粒子大小不变**：保持`particleSize`恒定
- **通过数量控制粗细**：宽的地方多层粒子并列
- **优点**：视觉更均匀，不会出现"大疙瘩"

### 5.3 叶子和果实仅在外层分支生成

**代码位置**：`consciousnessTreeGenerator.ts:212-220`

```typescript
// 叶子和果实（仅外层分支）
if (!isRoot && currentDepth > maxDepth * 0.6) {
  // 叶子：60%层级以上，根据leafDensity概率生成
  if (Math.random() < leafDensity) {
    addParticle(endX, endY, 'leaf')
  }
  // 果实：最外层，根据fruitProbability概率生成
  if (currentDepth === maxDepth && Math.random() < fruitProbability) {
    addParticle(endX, endY, 'fruit')
  }
}
```

**条件**：
- 叶子：层级 > `depth * 0.6` 且随机 < `leafDensity`
- 果实：层级 = `depth`（最外层）且随机 < `fruitProbability`
