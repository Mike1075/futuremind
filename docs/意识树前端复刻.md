  实际技术栈分析

  这个页面实现主要依赖：

  1. 原生技术 (无第三方依赖)

  - HTML5 Canvas API - 核心绘图引擎
  - React Hooks - useState, useRef, useEffect,
  useCallback
  - TypeScript - 类型系统
  - CSS-in-JS - 内联样式

  2. 样式框架

  - Tailwind CSS - 所有的 className 样式
  className="absolute top-6 right-6 space-y-3"
  className="block w-32 px-4 py-2 bg-white/90 
  hover:bg-white"

  3. 核心算法

  - Canvas 2D 绘图 - 手写的绘图引擎
  - 数学计算 - 贝塞尔曲线、向量运算、HSB颜色转换
  - 动画循环 - requestAnimationFrame

  npm install tailwindcss@^4 @tailwindcss/postcss@^4
  npm install typescript@^5 @types/react@^19
  @types/react-dom@^19

  关键特点

  这个实现的独特之处：
  1. 零第三方图形库 - 完全手写Canvas绘图
  2. 纯TypeScript算法 -
  所有生长、动画、交互逻辑都是原创
  3. 轻量级依赖 - 主要依赖React和Canvas原生API
  4. 高性能 - 90fps流畅动画，无外部渲染库

  这个页面的核心价值在于自研的算法系统，而不是依赖现
  成的图形库，这也是为什么代码有1200多行的原因。
