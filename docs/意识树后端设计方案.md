-- 1. 创建触发器函数
CREATE OR REPLACE FUNCTION public.handle_new_user_exploration_record()
RETURNS TRIGGER AS $$
BEGIN
    -- 在 user_domain_exploration 表中为新用户插入一行初始数据
    INSERT INTO public.user_domain_exploration (user_id)
    VALUES (new.id);
    RETURN new;
END;
$$ LANGUAGE plpgsql;

-- 2. 将触发器绑定到 profiles 表
CREATE TRIGGER on_profile_created
    AFTER INSERT ON public.profiles
    FOR EACH ROW EXECUTE PROCEDURE public.handle_new_user_exploration_record();
第二部分：边缘函数设计方案
边缘函数名称: evaluate-and-grow-tree

触发时机:

在您的应用后端逻辑中，监听gaia_conversations表。当检测到某个用户的messages数组长度达到10的倍数时，就调用此边缘函数。

输入 (Input):

{ "user_id": "用户的UUID" }

执行步骤 (Execution Steps):

获取双重上下文:

历史水平: SELECT domain_scores FROM public.user_domain_exploration WHERE user_id = [传入的user_id];

最新行为: SELECT messages FROM public.gaia_conversations WHERE user_id = [传入的user_id] ORDER BY created_at DESC LIMIT 1; 然后从messages JSON数组中取出最后20条消息。

构建AI提示词: 将上一步获取的domain_scores（历史水平）和20条messages（最新行为）填入我们之前设计的、包含上下文的“智能提示词”模板中。

调用AI大模型: 通过HTTPS请求将完整的提示词发送给OpenAI或任何其他大模型API。

接收并校验结果: 等待AI返回“成长增量”的JSON。务必校验返回的是否是合法的JSON，以及是否包含所有五个领域的_growth字段。

调用数据库函数: 调用名为 update_exploration_and_tree_view 的数据库函数(RPC)，并将user_id和AI返回的“成长增量”JSON作为参数传入。这是唯一与数据库发生写的操作。

第三部分：数据库函数 (RPC) 设计方案

这个函数是保证数据一致性的关键，它将所有写操作封装在一个原子事务中。

数据库函数名称: update_exploration_and_tree_view

目的: 接收AI评估出的成长增量，原子性地更新“事实数据源”（探索记录）和“视觉视图”（意识树）。

输入 (Input):

p_user_id UUID

p_growth_scores_json JSONB (例如: {"self_awareness_growth": 5, ...})

执行步骤 (Execution Steps):

在一个事务中自动执行以下所有操作:

更新“事实数据源” (user_domain_exploration):

读取该用户当前的domain_scores JSONB。

使用PostgreSQL的JSONB函数，将传入的p_growth_scores_json中的增量分数，累加到现有的depth_score上。

UPDATE该用户的domain_scores和last_evaluated_at字段。

重新生成“视觉视图” (consciousness_tree_view):

读取刚刚更新的、最新的总domain_scores。

根据最新的总分，应用视觉转换公式（例如 length = depth_score * 0.5）来计算roots部分的JSON数组。

(未来) 在这里也可以加入计算树干、枝叶等其他部分的逻辑。

将计算出的roots和其他部分组合成一个完整的consciousness_tree_view JSON对象。

更新profiles表:

UPDATE public.profiles SET consciousness_tree_view = [新生成的完整JSON] WHERE id = p_user_id;

这个设计方案为您提供了一个健壮、可扩展且逻辑清晰的框架，完美地实现了您关于“连续成长过程”的深刻见解。
