## 🔧 前端集成

### 1. API使用示例

```typescript
import consciousnessTreeAPI from '@/lib/api/consciousness-tree'

// 获取意识树视图
```

```
用户对话 → 盖亚API保存消息 → 消息数量检查 → 触发AI评估
    ↓
OpenAI分析对话内容 → 生成五领域成长分数 → 更新探索记录
    ↓
重新计算意识树视图 → 更新profiles表 → 前端刷新显示
```
