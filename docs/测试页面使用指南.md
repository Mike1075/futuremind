## 📖 功能说明

这是一个**完整的Web测试界面**，用于测试 `summarize-user-activity` Edge Function。

### ✨ 核心功能

1. **自动加载测试用户** - 列出所有有数据的学生用户
2. **实时日志显示** - 查看每一步执行过程
3. **维度选择** - 灵活测试对话/作业/项目任意维度
4. **AI结果展示** - 美观展示GPT-5 Mini生成的总结
5. **数据库验证** - 自动验证数据是否正确存储

## 🐛 常见问题排查

### Q1: 页面显示"正在加载用户..."但一直不出现用户列表

**原因**: 数据库中没有学生用户或所有学生都没有数据

**解决**:
1. 检查数据库中是否有 `role = 'student'` 的用户
2. 检查这些用户是否有对话/作业/项目数据

**SQL查询验证**:
```sql
SELECT COUNT(*) FROM profiles WHERE role = 'student';
```

### Q2: 点击"开始测试"后报错 "请先选择一个测试用户"

**原因**: 未选择用户

**解决**: 点击左侧用户列表中的任意用户卡片

### Q3: 调用失败，显示 "HTTP 401: ..."

**原因**: 认证失败

**解决**:
1. 确保您已登录系统
2. 检查 `NEXT_PUBLIC_SUPABASE_ANON_KEY` 环境变量是否正确配置

### Q4: AI总结内容为空或显示"无法生成总结"

**原因**: OpenAI API调用失败

**解决**:
1. 检查 Edge Function 日志：Supabase Dashboard → Functions → summarize-user-activity → Logs
2. 验证 `OPENAI_API_KEY` 环境变量是否正确
3. 检查 OpenAI 账户余额

### Q5: 数据库验证失败

**原因**: 数据未正确存储到 `student_summaries` 表

**解决**:
1. 查看 Edge Function 日志中的详细错误信息
2. 手动查询数据库:
```sql
SELECT * FROM student_summaries WHERE user_id = '你的用户ID';
```

## 🎨 页面功能详解

### 1. 测试用户列表

- **自动加载**: 页面打开时自动查询所有有数据的学生
- **数据统计**: 显示每个用户的对话、作业、项目数量
- **选中状态**: 选中的用户卡片会高亮显示（绿色边框）

### 2. 维度选择

支持任意组合：
- 只测试对话维度
- 只测试作业维度
- 只测试项目维度
- 测试全部三个维度
- 测试任意两个维度组合

### 3. 执行日志

- **实时更新**: 每个步骤都会立即显示
- **时间戳**: 精确到毫秒，方便性能分析
- **颜色编码**: 不同类型的日志用不同颜色区分
- **自动滚动**: 新日志会自动滚动到可见区域

### 4. 结果展示

- **分维度展示**: 每个维度独立显示，方便查看
- **颜色区分**:
  - 对话维度 - 蓝色
  - 作业维度 - 黄色
  - 项目维度 - 紫色
- **统计数据**: 显示对话数、作业数等关键指标

## 🔧 开发者选项

### 查看Edge Function日志

在 Supabase Dashboard 中：
1. Functions → summarize-user-activity
2. 点击 "Logs" 标签
3. 查看详细的服务器端日志

### 查看数据库数据

```sql
-- 查看最新的总结记录
SELECT * FROM student_summaries
ORDER BY generated_at DESC
LIMIT 5;

-- 查看特定用户的总结
SELECT
  course_summaries->'dialogue'->>'summary' as dialogue_summary,
  course_summaries->'coursework'->>'summary' as coursework_summary,
  course_summaries->'projects'->>'summary' as projects_summary
FROM student_summaries
WHERE user_id = '用户ID';
```

### 手动调用API

使用 curl 或 Postman:
```bash
curl -X POST \
  'https://lvjezsnwesyblnlkkirz.supabase.co/functions/v1/summarize-user-activity' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "userId": "b9a9ab9d-2978-4918-80e0-d12422e24cb2",
    "dimensions": ["dialogue"]
  }'
```

## 📈 性能指标

正常执行时间：
- **单个维度**: 3-6 秒
- **两个维度**: 6-10 秒
- **三个维度**: 10-15 秒

如果超过30秒，请检查：
1. 网络连接
2. OpenAI API 响应速度
3. 数据量是否过大

1. **先测试单个维度** - 验证功能正常后再测试全部维度
