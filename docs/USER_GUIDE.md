## 一、注册与登录

### 方式一：首页登录

1. 打开 **www.futuremind2075.com**
2. 点击左上角 **「登录」** 按钮
3. 在弹窗中输入邮箱和密码
4. 点击「登录」

### 方式二：登录页面

1. 直接访问 **www.futuremind2075.com/login**
2. 输入邮箱和密码登录

### 注册新账号

1. 在登录界面点击 **「还没有账号？立即注册」**
2. 填写：
   - 姓名（显示名称）
   - 邮箱（用于登录）
   - 密码（需8位以上，包含大小写字母和数字）
3. 点击「注册」
4. 去邮箱查收验证邮件，点击链接完成验证

### 忘记密码

1. 在登录界面点击 **「忘记密码？」**
2. 输入注册邮箱
3. 点击「发送重置邮件」
4. 去邮箱点击重置链接，设置新密码

首页有以下主要入口：

## 三、学生使用指南

### 3.1 个人门户

访问路径：首页 → 「个人门户」按钮

门户页面包含：

**左侧 - 我的课程**
- 显示老师分配给你的课程
- 点击课程卡片进入学习
- 每个课程显示学习进度百分比
- 点击「探索所有课程」可查看全部课程

**右侧 - 意识进化树**
- 你的学习成长可视化图
- 点击「查看完整意识树」看大图

### 3.2 学习课程

平台提供三大课程体系：

#### 1. 自在聆听·观音之旅

14天的聆听练习课程，通过克里希那穆提的智慧引导，培养觉察能力：

1. 按顺序学习，每天一课
2. 播放音频，跟随引导冥想
3. 可以记录每天的学习感受
4. 建议每天固定时间练习

#### 2. 欢迎来到地球

基于《欢迎来到地球》纪录片的6阶段探索课程：

1. 进入课程后，按阶段顺序学习
2. 每个阶段包含多个课程内容
3. 学习内容包括：
   - 观看纪录片视频
   - 阅读知识点解读
   - 思考启发式问题
   - 完成「小探险家项目」（动手实践任务）
4. 完成任务后提交作业，AI会即时批改

#### 3. 伊卡洛斯计划

PBL（项目制学习）课程体系，培养探索精神和实践能力：

1. 进入课程后，会看到项目地图（三角形/圆形排列）
2. 项目按模块分类，如：
   - 无形的纽带（动物与自然）
   - 生命的奥秘
   - 创意表达
3. 选择感兴趣的项目加入
4. 每个项目包含：
   - 项目介绍和目标
   - 周计划任务
   - 阶段性成果提交
5. 可以查看「我的项目」跟踪进度
6. 完成项目任务后提交成果

**伊卡洛斯项目特点**：
- 项目有难度等级标识
- 可以同时参与多个项目
- 鼓励动手实践和创造

### 3.3 提交作业

1. 在课程详情页找到任务区域
2. 点击「提交作业」
3. 输入文字内容（必填）
4. 如需上传图片，点击上传按钮
5. 可选择是否「公开作业」（让其他同学看到）
6. 点击「提交」
7. **AI会即时批改**，几秒后显示评分和反馈

**AI批改说明**：
- 提交后AI会立即给出评分（0-100分）和详细反馈
- 60分以上为及格，可解锁下一课
- 低于60分需要重新提交改进后的作业
- 评分会影响你的意识树成长

**注意**：
- 作业可以多次提交，系统保留最新一次
- 选择「公开」的优秀作业会展示给其他同学

### 3.4 与盖亚对话

「盖亚」是你的AI学习伙伴：

1. 点击页面右下角的蓝色圆形按钮
2. 输入问题或想法
3. 盖亚会根据你的学习情况给出回答

适合问的问题：
- 对课程内容的疑问
- 学习上遇到的困惑
- 想要深入探讨的话题

### 3.5 探索者联盟

这是项目协作平台：

1. 从首页点击「探索者联盟」进入
2. 你会看到「我的组织」列表：
   - **社区项目**：公开的协作项目
   - **我的项目**：个人项目空间
3. 点击组织进入，可以：
   - 查看和加入项目
   - 与项目成员交流
   - 参与任务协作

## 四、老师使用指南

### 4.1 进入管理后台

- 首页点击「管理后台」按钮
- 或直接访问 **www.futuremind2075.com/admin**

### 4.2 课程管理

路径：管理后台 → 课程管理

可以管理三大课程体系：
- **自在聆听·观音之旅** - 14天冥想课程
- **欢迎来到地球** - 6阶段纪录片课程
- **伊卡洛斯计划** - PBL项目制学习

每个课程页面左侧有两个入口：
- **选课学员** - 管理哪些学员可以学习此课程
- **课程分组** - 管理学习小组

右侧是**课程内容编辑区**。

> **说明**：作业由AI即时批改，老师无需手动批改。

### 4.3 编辑课程内容

以「欢迎来到地球」为例：

1. 进入课程管理 → 欢迎来到地球
2. 左侧选择要编辑的阶段
3. 右侧可以编辑：
   - 阶段标题和副标题
   - 主纪录片视频链接
   - 知识点列表
   - 思考问题（观看前/中/后）
   - 小探险家项目
   - 补充视频资源

4. 编辑完成后点击「保存」

### 4.4 管理选课学员

1. 进入课程页面，点击左侧「选课学员」
2. 查看当前选课学员列表
3. 点击「添加学员」可以添加新学员
4. 点击学员右侧的删除按钮可以移除学员

### 4.5 管理学习小组

1. 进入课程的「分组」页面
2. 点击「创建分组」
3. 输入分组名称
4. 添加成员
5. 保存

分组类型：
- **课程内分组**：只在该课程内有效
- **全局分组**：跨所有课程有效

## 五、校长使用指南

校长拥有老师的所有权限，另外还可以：

### 5.1 人员管理

路径：管理后台 → 人员管理

页面顶部有4个统计卡片可切换显示：
- **全部人员** - 所有用户
- **校长** - 校长角色
- **教师** - 教师角色
- **学员** - 学员角色

**修改用户角色**：
1. 在用户列表找到目标用户
2. 在「操作」列的下拉框中选择新角色（校长/教师/学员）
3. 确认修改

**按邮箱设置角色**：
1. 点击右上角「设置角色」按钮
2. 输入已注册用户的邮箱
3. 选择要设置的角色（教师或学员）
4. 点击确定

> **注意**：用户必须先注册账号，才能在此处设置其角色。

### 5.2 盖亚知识库管理

路径：管理后台 → 盖亚知识库

可以管理AI助手「盖亚」的知识来源：
- 上传知识文档
- 管理知识分类
- 查看知识库状态

## 六、常见问题

### Q: 注册后没收到验证邮件？
A: 请检查垃圾邮件文件夹。如仍未收到，等待几分钟后重新注册。

### Q: 登录后看不到课程？
A: 需要老师或校长为你分配课程。请联系你的老师。

### Q: 如何修改个人资料？
A: 登录后点击左上角用户名 → 个人资料 → 编辑保存。

### Q: 作业提交后可以修改吗？
A: 可以重新提交，新提交会覆盖旧的。AI会重新批改新提交的内容。

### Q: 盖亚回复很慢怎么办？
A: AI回复需要3-10秒，请耐心等待。超过30秒可刷新页面重试。

### Q: 手机可以使用吗？
A: 可以！网站完全支持手机浏览器，体验一致。

### Q: 探索者联盟是做什么的？
A: 这是项目协作平台，可以：
- 加入公开项目与他人协作
- 创建自己的项目
- 与项目成员交流讨论

### Q: 如何联系管理员？
A: 请通过您报名时的联系方式联系管理员。
