## 🎨 使用Lovart的步骤

### 1. 注册Lovart账号
- 访问 https://www.lovart.ai
- 注册免费账号（可能需要排队等候）
- 登录后进入创作界面

### 2. 批量生成模式
- 选择 "Batch Mode"（批量模式）
- 可一次生成最多40个变体
- 推荐每次生成一个Level的5个阶段

### 3. 参数设置
```
Model: Flux (最新模型，质量最高)
Style: Watercolor Illustration
Size: 1024 x 1024 (正方形)
Quality: High
Seed: Random (每个阶段可以固定seed保持风格一致)
```

### 4. 导出设置
```
Format: PNG
Background: Transparent
Resolution: 300 DPI (用于高质量展示)
```

## 📝 通用提示词模板

### 核心要素

每个提示词都包含以下元素：
1. **主体描述** - 树的形态（seed/sprout/seedling/young/mature）
2. **颜色主题** - 对应脉轮颜色（red/orange/yellow/green/cyan/blue/purple）
3. **风格指定** - watercolor illustration, organic shapes
4. **细节要求** - roots, trunk, canopy, leaves, fruits
5. **氛围营造** - magical, spiritual, glowing particles
6. **技术要求** - transparent background, 4K/8K resolution

### 负面提示词（通用）

```
Negative Prompt (All Stages):
geometric shapes, perfect circle, straight lines, stiff,
cartoon character, childish drawing, stick figure,
dotted lines, pearl string effect, symmetrical,
overly simple, realistic photography, 3D render,
busy background, cluttered, text, watermark
```

## ✅ 质量控制建议

### 检查清单

生成每张图后检查：
- [ ] 背景是透明的
- [ ] 颜色符合脉轮主题
- [ ] 没有几何感（圆形、直线）
- [ ] 根系是流畅曲线，不是珠子串
- [ ] 树冠是蓬松云朵形，不是完美圆形
- [ ] 整体风格是水彩插画，不是卡通
- [ ] 图片分辨率达到4K (Seed-Young) 或 8K (Mature)
- [ ] 没有文字水印

### 如果效果不理想

1. **根系太像珠子串**:
   - 在提示词中强调："smooth flowing brush strokes, NO dotted lines"
   - 增加负面提示："pearl necklace, beaded line"

2. **树冠太圆**:
   - 强调："irregular cloud shape, scalloped edges, NOT geometric circle"
   - 参考词："like fluffy clouds, cotton candy texture"

3. **整体太卡通**:
   - 强调："professional watercolor painting, sophisticated art"
   - 避免："cute, cartoon, clipart"

4. **颜色不对**:
   - 直接在提示词开头说明："Color theme: #HEX_CODE"
   - 使用颜色名："deep crimson red, NO pink, NO orange"

### 批量生成策略

**推荐流程**:
1. **第一轮**: 生成Level 1的所有5个阶段，确定满意的风格
2. **固定Seed**: 使用相同的seed值生成Level 2-7，保持风格一致
3. **微调**: 对于不满意的单独重新生成
4. **最终筛选**: 从每个阶段的多个变体中选择最佳版本
